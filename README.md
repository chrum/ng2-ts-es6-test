# ng2-ts-es6-test

**Simple (aspiring) ng2 seed with both ts and es6 files**

```bash
# Clone this repository
git clone https://github.com/chrum/ng2-ts-es6-test
# Go into the repository
cd ng2-ts-es6-test
# Install dependencies and run the app
npm install && npm start

```
